### 💡 **About Weather App**

The **Weather Dashboard App** is a web application that allows users to:
- 🔍 Search for real-time weather data for any location worldwide  
- 📊 Get historical weather data for specific coordinates  
- 🌡️ View data like temperature, humidity, wind speed, etc.

🛠 Built using **React.js** and styled with **Material-UI**, it integrates the **OpenWeather API** for real-time and historical data up to 45 years. The app is fully responsive and works smoothly on both desktop and mobile devices.

Feel free to explore the app and get a glimpse of how the weather changes around the world! 🌍
